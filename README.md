# react 安装
* 1、安装初始化工具
```bash
npm install -g create-react-app
```
* 2、初始化react项目（已经安装配置好babel和webpack）
```bash
create-react-app my-react
```
* 3、启动react demo
```bash
cd my-react
npm start
```
* 4、打包生成生产环境
```bash
npm run build
```
